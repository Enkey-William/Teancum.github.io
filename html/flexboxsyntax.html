Flexbox Syntax

Why Flexbox?

Flexbox is a one-dimentional system put in place (using CSS3) to help organize web page layouts. It is relatively simplistic yet strict in application (just like all CSS), which may cause confusion when first put into practice.

It is meant to be used to help simplify CSS and strengthen a web designers ability to make responsive internet sites.

How do I use it?

It is very simple. First, a container must be set to 'display: flex' or 'display: inline-flex'. This is very important, because the Flexbox is triggered into action by the 'display' property. Flexbox then uses typical CSS syntax:

property: value

which is placed in an element selector, such as .containerclass, #boxid, et cetera. There are different properties and values which can be used with parent and child selectors in order to get the desired effect.

Properties: Values

Flex can be initiated by the 'display' property (ie a container); flex or inline-flex are the values. The following can then be used in the container element selector to influence children elements:

  • flex-direction: row, row-reverse, column, column-reverse; row is the default.
  • flex-wrap: nowrap, wrap, wrap-reverse; nowrap is the default.
  • flex-flow: [flex-direction flex-wrap]; a combination of the first two.
  • justify-content: flex-start, flex-end, center, space-between, space-around; affects content along main axis.
  • align-items: flex-start, flex-end, center, baseline, stretch; affects content along cross axis.
  • align-content: stretch, flex-start, flex-end, center, space-around, space-between; affects space along cross axis (when there is more than one row/column); stretch is the default;

The following properties are placed within the children element selectors:

  • order: (integer value); the default is source order.
  • flex-grow: (integer value); the default is 0.
  • flex-shrink: (integer value); positive integers only.
  • flex-basis: auto, integer value; the default is 0.
  • flex: [flex-grow, flex-shrink, flex-basis]
  • align-self: flex-start, flex-end, center, baseline, stretch.
