HTML and CSS RULES

HTML Syntax Rules

  1. Elements and attributes are lower case.
  2. Attribute values are quoted.
  3. All elements must close.
  4. Elements must open and close in order.

CSS Syntax Rules

  1. Rule Set
    • Selector
    • Declaration Block
      • Property: part before colon
      • Value: part after the colon
  3. Selector examples
    • * - Universal Selector
    • Element Type Selector (Type Selector) - for <nav>, <h1>, etc
    • ID Selector - #[idname]
    • Class Selector - .classname (just a dot or period followed [no spaces] by the class name)
  5. Family Trees
    • Parent, child, descendant, and sibling elements
    • Combinators are used to group families together (selector space selector; .box #container)
    • A child combinator will only affect immediate descendants (.box > #container)
    • Sibling combinators affect elements beside each other in a larger container (h1 ~ p)
    • Immediate sibling only affects elements adjacent (p + p)
  7. Attribute Selector (input[type])
  8. Pseudo-Class (a:hover)
  9. Pseudo-Element (.box:before)
  10. Elements can be combined (p.box OR p, div, .box OR etc)
  11. Cascade: the lower down an element, the higher it's precedence
  12. Specificity
    • Specificity calculations example
    • 1 point - element
    • 10 points - class
    • 100 points - id

Box Model

  • Content - hold the content
  • Padding - inside the element around the content
  • Border - at the edge of the element around the padding
  • Margin - outside the element surrounding the border
